# User-s-Guide-Program-DiDePT
This is a guide to handle the data analysis software of the DiDePT
Before running the program it must be installed the language program Python 3.13.2 on the PC, which can be downloaded in: https://www.python.org/downloads/.
Firstly, download the folder "DiDePT Proccessing", in this you will find the necessary python programs to start the program. Please, put your image folder inside "DiDePT Proccesing" folder once you have the images.
